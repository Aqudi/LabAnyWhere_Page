# LabAnyWhere_Page

## 사용한 도구
+ Django rest framework
+ React

## 필요한 모듈 설치
+ pip install -r requirements.txt
+ yarn 또는 npm install

## 로컬 실행 방법
1. 장고 서버 실행 
    ```
    python manage.py runserver
    ```
2. React 개발서버 실행
    ```
    yarn start
    ```
    또는
    ```
    npm run start
    ```
3. Recat 개발서버 localhost:3000
   Rest 서버      localhost:8000
